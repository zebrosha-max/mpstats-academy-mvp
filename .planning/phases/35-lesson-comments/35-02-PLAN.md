Create 3 component files in `apps/web/src/components/comments/`:

**CommentInput.tsx:**
- Props: `{ lessonId: string, parentId?: string, onCancel?: () => void, autoFocus?: boolean, onSuccess?: () => void }`
- Uses shadcn `Textarea` with placeholder "Напишите комментарий..." (or "Напишите ответ..." when parentId provided)
- Character counter: `{content.length} / 1500` displayed below textarea, text turns red (`text-destructive`) when > 1500
- "Отправить" Button (default variant) disabled when content is empty or > 1500
- "Отмена" Button (ghost variant) shown only when `onCancel` prop provided (reply mode)
- Uses `trpc.comments.create.useMutation` with optimistic update:
  - `onMutate`: call `utils.comments.list.setInfiniteData` to insert optimistic comment
  - `onError`: revert via saved context, show `toast.error("Не удалось отправить комментарий. Попробуйте ещё раз.")`
  - `onSettled`: invalidate `utils.comments.list`
  - On success: clear textarea, call `onSuccess?.()` and `onCancel?.()` if reply
- Textarea auto-grows from 2 rows to max 5 rows (use `rows={2}` and CSS `min-h-[5rem] max-h-[10rem] resize-none`)
- Layout: textarea full width, below it a flex row with character counter on left, buttons on right

**CommentItem.tsx:**
- Props: `{ comment: CommentData, currentUserId: string, currentUserRole: string, isReply?: boolean }`
- Avatar: `w-8 h-8` for root, `w-7 h-7` for reply. Use `<img>` with `rounded-full object-cover ring-2 ring-mp-gray-100`. Fallback: gradient div `bg-gradient-to-br from-mp-blue-500 to-mp-blue-600` with white initials (first 2 chars of `comment.user.name` or "?")
- Author name: `text-sm font-semibold text-mp-gray-900`
- Separator dot: `text-mp-gray-300` between name and timestamp
- Relative timestamp function `formatRelativeTime(date: Date)`: "только что" (<1 min), "N мин назад" (<60 min), "N ч назад" (<24h), "вчера" (<48h), else "DD MMM" (e.g. "15 мар"). Use Russian month abbreviations: ["янв","фев","мар","апр","май","июн","июл","авг","сен","окт","ноя","дек"]
- Comment text: `text-sm text-mp-gray-700 whitespace-pre-wrap` (plain text, line breaks preserved per UI-SPEC)
- Action row for root comments only (not replies):
  - "Ответить" button: `text-xs text-mp-gray-400 hover:text-mp-blue-600` — on click, sets `showReplyInput` state to true
  - Trash icon button: visible when `comment.userId === currentUserId` or `currentUserRole === 'ADMIN'` or `'SUPERADMIN'` — `text-mp-gray-400 hover:text-destructive`
- Reply action row: only trash icon (no "Ответить" per D-06, 1-level only)
- Delete uses AlertDialog: title "Удалить комментарий?", description "Это действие нельзя отменить.", actions "Удалить" (destructive) / "Отмена"
- Delete mutation: `trpc.comments.delete.useMutation` with optimistic removal from list and `toast.error` on failure
- Replies rendered inline below root comment with `ml-6 border-l-2 border-mp-gray-200 pl-4 space-y-3` (per UI-SPEC reply indent)
- When `showReplyInput` is true, render `<CommentInput>` after replies with `parentId={comment.id}`, `autoFocus`, `onCancel` to hide

**CommentSection.tsx:**
- Props: `{ lessonId: string }`
- Uses `trpc.comments.list.useInfiniteQuery({ lessonId }, { getNextPageParam: (last) => last.nextCursor })`
- Section header: `<h3 className="text-base font-semibold text-mp-gray-900">Комментарии ({totalCount})</h3>` where totalCount from first page data
- Top input: `<CommentInput lessonId={lessonId} />` for new root comments
- Fetches current user profile via `trpc.profile.get.useQuery()` for `currentUserId` and `currentUserRole`
- Comment list: map over `data.pages.flatMap(p => p.comments)`, render `<CommentItem>` for each
- Empty state (when totalCount === 0 and no pages): centered div with speech bubble SVG icon (inline, `w-12 h-12 text-mp-gray-300`), heading "Пока нет комментариев" (text-sm font-medium text-mp-gray-500), body "Будьте первым — поделитесь мыслями по уроку" (text-xs text-mp-gray-400)
- Loading state: 3 skeleton rows — each with `Skeleton` circle (w-8 h-8 rounded-full) + two `Skeleton` text bars
- "Показать ещё" button: `<Button variant="outline" size="sm">` at bottom, visible when `hasNextPage`, onClick calls `fetchNextPage`, shows spinner while `isFetchingNextPage`
- Background: `bg-mp-gray-50 rounded-lg p-4` wrapper for the entire section
- `data-tour="lesson-comments"` attribute on the outermost wrapper div (for Phase 36 product tour)
