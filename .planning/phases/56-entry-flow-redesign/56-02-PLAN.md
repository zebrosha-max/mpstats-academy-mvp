| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-56-04 | Tampering | onboarding.complete input | mitigate | `z.enum([...])` whitelist на marketplaces/goals/experienceLevel — неизвестные ключи отклоняются до записи в БД. Unit-тест Task 2 покрывает отклонение. |
