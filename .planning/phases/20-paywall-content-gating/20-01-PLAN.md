```typescript
const FREE_LESSON_THRESHOLD = 2; // lessons with order <= 2 are free
```

**Three exports:**

a) `getUserActiveSubscriptions(userId, prisma)` -- fetches all user subscriptions with status IN ['ACTIVE', 'CANCELLED'] AND currentPeriodEnd > now, includes plan relation. Single query, returns array.

b) `isLessonAccessible(lesson: { order: number; courseId: string }, subscriptions, billingEnabled)` -- synchronous, pure function. Returns boolean. Logic order:
   - !billingEnabled -> true
   - lesson.order <= FREE_LESSON_THRESHOLD -> true
   - any subscription with plan.type === 'PLATFORM' -> true
   - any subscription with plan.type === 'COURSE' AND courseId matches -> true
   - else false

c) `checkLessonAccess(userId, lesson: { order: number; courseId: string }, prisma)` -- async wrapper. Calls isFeatureEnabled('billing_enabled'), if false returns { hasAccess: true, reason: 'billing_disabled', hasPlatformSubscription: false }. Otherwise calls getUserActiveSubscriptions + isLessonAccessible. Returns `AccessResult`:

```typescript
interface AccessResult {
  hasAccess: boolean;
  reason: 'free_lesson' | 'platform_subscription' | 'course_subscription' | 'billing_disabled' | 'subscription_required';
  hasPlatformSubscription: boolean;
}
```

Export the AccessResult type as well.

Import `isFeatureEnabled` from `./feature-flags` and PrismaClient type from `@mpstats/db`.
