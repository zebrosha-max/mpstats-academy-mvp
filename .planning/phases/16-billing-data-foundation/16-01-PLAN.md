1. `SubscriptionPlan` -- defines plan terms (separate model so owner can edit prices via admin without deploy):
   - `id String @id @default(cuid())`
   - `type SubscriptionType @unique` -- one plan per type (COURSE or PLATFORM)
   - `name String` -- display name ("Подписка на курс" / "Полный доступ")
   - `price Int` -- rubles, e.g. 4990
   - `intervalDays Int @default(30)` -- billing cycle
   - `isActive Boolean @default(true)` -- can be disabled
   - `createdAt DateTime @default(now())`
   - `updatedAt DateTime @updatedAt`
   - `subscriptions Subscription[]` -- reverse relation

2. `Subscription`:
   - `id String @id @default(cuid())`
   - `userId String`
   - `planId String`
   - `courseId String?` -- null for PLATFORM, course ID for COURSE
   - `status SubscriptionStatus @default(ACTIVE)`
   - `currentPeriodStart DateTime`
   - `currentPeriodEnd DateTime`
   - `cancelledAt DateTime?`
   - `createdAt DateTime @default(now())`
   - `updatedAt DateTime @updatedAt`
   - Relations: `user UserProfile @relation(fields: [userId], references: [id], onDelete: Cascade)`, `plan SubscriptionPlan @relation(fields: [planId], references: [id])`, `course Course? @relation(fields: [courseId], references: [id])`
   - `payments Payment[]` reverse relation
   - Indexes: `@@index([userId])`, `@@index([userId, status])`, `@@index([courseId])`

3. `Payment`:
   - `id String @id @default(cuid())`
   - `subscriptionId String`
   - `amount Int` -- rubles
   - `status PaymentStatus @default(PENDING)`
   - `cloudPaymentsTxId String? @unique` -- for Phase 18 idempotency, nullable now
   - `paidAt DateTime?`
   - `createdAt DateTime @default(now())`
   - Relations: `subscription Subscription @relation(...)`, `events PaymentEvent[]`
   - Indexes: `@@index([subscriptionId])`, `@@index([cloudPaymentsTxId])`

4. `PaymentEvent`:
   - `id String @id @default(cuid())`
   - `paymentId String`
   - `type String` -- raw event type from CloudPayments ("pay", "fail", etc.)
   - `payload Json` -- full webhook payload for audit
   - `createdAt DateTime @default(now())`
   - Relations: `payment Payment @relation(..., onDelete: Cascade)`
   - Index: `@@index([paymentId])`

   - `enabled Boolean @default(false)`

   - `updatedAt DateTime @updatedAt`

```typescript
import { prisma } from '@mpstats/db';

/**
 * Check if a feature flag is enabled.
 * Returns false if the flag doesn't exist (safe default).
 */
export async function isFeatureEnabled(key: string): Promise<boolean> {
  const flag = await prisma.featureFlag.findUnique({
    where: { key },
  });
  return flag?.enabled ?? false;
}
```
