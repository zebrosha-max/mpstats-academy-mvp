**2. Create `generateSectionedPath()` function in `diagnostic.ts`** (new function alongside existing `generateFullRecommendedPath`):

```typescript
import type { SectionedLearningPath, LearningPathSection } from '@mpstats/shared';

async function generateSectionedPath(
  prisma: PrismaClient,
  skillProfile: SkillProfile,
  sessionId: string,
  answers: Array<{ isCorrect: boolean; sourceData: any; skillCategory: string }>,
): Promise<SectionedLearningPath>
```

**Algorithm (exact implementation from RESEARCH Pattern 4):**

a) **Section 1 "Errors" (Проработка ошибок):**
- `wrongAnswers = answers.filter(a => !a.isCorrect && a.sourceData?.lessonIds?.length > 0)`
- `errorLessonIds = [...new Set(wrongAnswers.flatMap(a => a.sourceData.lessonIds))]`
- Query lessons by IDs, sort by: category weakness (look up `skillProfile[category]`, lowest first), then `lesson.order` within same category
- Build hints array from wrongAnswers: for each errorLessonId, find all wrong answers that reference it, extract questionText (from session questions Json) and timecodes

b) **Section 2 "Deepening" (Углубление):**
- `weakCategories` = categories where score < 70 (use `SkillProfile` keys: analytics, marketing, content, operations, finance)
- Query lessons where `skillCategories` Json array overlaps with weakCategories AND `id NOT IN errorLessonIds`
- Use raw SQL or Prisma `JsonFilter` — since `skillCategories` is Json, use: `prisma.lesson.findMany()` then filter in JS: `lessons.filter(l => { const cats = l.skillCategories as string[] | null; return cats?.some(c => weakCategories.includes(c)); })`
- Sort by category weakness then `order`

c) **Section 3 "Growth" (Развитие):**
- `midCategories` = categories where score >= 70 AND score <= 85
- Same pattern: query all lessons, filter by skillCategories overlap, exclude lessons already in sections 1+2
- Sort by `order`

d) **Section 4 "Advanced" (Продвинутый уровень):**
- `strongCategories` = categories where score > 85
- Filter: `skillLevel === 'HARD'` AND skillCategories overlap with strongCategories AND not in sections 1+2+3
- Sort by `order`

e) Return `SectionedLearningPath`:
```typescript
return {
  version: 2,
  sections: [
    { id: 'errors', title: 'Проработка ошибок', description: `${errorLessons.length} уроков по темам, где были ошибки`, lessonIds: errorLessonIds, hints },
    { id: 'deepening', title: 'Углубление', description: `${deepeningLessons.length} уроков для слабых навыков`, lessonIds: deepeningIds },
    { id: 'growth', title: 'Развитие', description: `${growthLessons.length} уроков для средних навыков`, lessonIds: growthIds },
    { id: 'advanced', title: 'Продвинутый уровень', description: `${advancedLessons.length} уроков повышенной сложности`, lessonIds: advancedIds },
  ].filter(s => s.lessonIds.length > 0), // hide empty sections
  generatedFromSessionId: sessionId,
};
```

If `parsed` is `SectionedLearningPath` (has `version: 2`), return sectioned data:
```typescript
if (!Array.isArray(parsed) && parsed.version === 2) {
  // Collect all lessonIds from all sections
  const allLessonIds = parsed.sections.flatMap(s => s.lessonIds);

  // Fetch lessons with progress (same query as before but for sectioned IDs)
  const lessons = await ctx.prisma.lesson.findMany({
    where: { id: { in: allLessonIds } },
    include: {
      progress: { where: { path: { userId: ctx.user.id } } },
      course: { select: { title: true } },
    },
  });
  const lessonMap = new Map(lessons.map(l => [l.id, l]));

  // Build sections with full lesson data
  const sectionsWithData = parsed.sections.map(section => ({
    ...section,
    lessons: section.lessonIds
      .map(id => lessonMap.get(id))
      .filter(Boolean)
      .map(l => {
        const locked = !isLessonAccessible({ order: l!.order, courseId: l!.courseId }, subs, billingEnabled);
        return {
          id: l!.id,
          courseId: l!.courseId,
          courseName: l!.course.title,
          title: l!.title,
          description: l!.description,
          videoUrl: l!.videoUrl || '',
          videoId: l!.videoId,
          duration: l!.duration || 0,
          order: l!.order,
          skillCategory: l!.skillCategory,
          skillLevel: l!.skillLevel,
          status: (l!.progress[0]?.status || 'NOT_STARTED') as string,
          watchedPercent: l!.progress[0]?.watchedPercent || 0,
          locked,
        };
      }),
  }));

  const allLessonsFlat = sectionsWithData.flatMap(s => s.lessons);
  const completedCount = allLessonsFlat.filter(l => l.status === 'COMPLETED').length;

  return {
    generatedAt: path.generatedAt,
    sections: sectionsWithData,
    lessons: allLessonsFlat, // flat list for backward compat
    totalLessons: allLessonsFlat.length,
    completedLessons: completedCount,
    hasPlatformSubscription: subs.some(s => s.plan.type === 'PLATFORM'),
    isSectioned: true,
  };
}
```

If `parsed` is `string[]` (old format), keep existing behavior unchanged (return flat list with `isSectioned: false`).
