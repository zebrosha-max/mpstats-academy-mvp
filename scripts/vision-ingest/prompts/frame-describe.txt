Ты анализируешь кадр из видео-урока MPSTATS Academy (контент про маркетплейсы Wildberries / Ozon, аналитику, нейросети, контент-маркетинг).

Верни СТРОГО JSON-объект (без markdown-блоков, без пояснений до или после):

{
  "type": "slide" | "screen" | "talking_head" | "diagram" | "code" | "other",
  "summary": "1-2 предложения по-русски — что показано на экране и в чём суть для слушателя урока",
  "extracted": {
    "urls": ["..."],         // опционально: видимые URL/домены
    "numbers": ["..."],      // опционально: ключевые числа/метрики/проценты с экрана
    "tools": ["..."],        // опционально: названия инструментов/сервисов (MPSTATS, ChatGPT, Photoshop и т.п.)
    "other": ["..."]         // опционально: важный экранный текст не попавший в категории выше
  }
}

Правила:
- Если кадр — только говорящая голова без слайда/экрана/доски, верни type=talking_head, summary="Говорящая голова, нет визуального контента", все extracted-массивы пустые.
- Если виден интерфейс приложения/сайта/dashboard — type=screen, опиши какой раздел и что в нём делает спикер.
- Если виден слайд презентации — type=slide, передай тезис заголовка и 2-3 ключевые мысли.
- Если виден код / редактор — type=code, кратко опиши что за код.
- numbers — только осмысленные данные (цены, проценты, объёмы), не таймкоды плеера и не системные часы.
- НЕ выдумывай: если что-то нечитаемо — не вписывай.
- Краткость важнее: summary ≤ 200 символов.
