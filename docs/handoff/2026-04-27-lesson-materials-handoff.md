## Что появилось на платформе

Под видео уроков (на странице `/learn/[id]`) теперь есть новая секция **«Материалы к уроку»** — карточки с презентациями, таблицами расчётов, чек-листами, памятками и ссылками на сервисы.

- Юзеру с активной подпиской на курс — секция видна, кликает по карточке → скачивает файл / открывается сервис в новой вкладке
- Юзеру без подписки — секции вообще нет в интерфейсе урока (как будто её не существует)
- В Yandex Metrika летят события `MATERIAL_OPEN` (клик по карточке) и `MATERIAL_SECTION_VIEW` (когда секция попала в видимую область экрана)

## Как работать в новом разделе админки

### Где находится

Зайти на `https://platform.mpstats.academy/admin/content/materials` или через сайдбар: **Content → Materials** (новый пункт между Content и Comments). Доступ только у ADMIN и SUPERADMIN.

### Главная страница

Что видите:
- Заголовок «Материалы к урокам»
- Кнопка **«+ Добавить материал»** (правый верхний угол)
- Блок **«Фильтры»**:
  - 5 чипов по типу материала (Презентация / Таблица расчётов / Внешний сервис / Чек-лист / Памятка) — клик включает/выключает фильтр
  - Поиск по названию материала
  - Выпадашка «Все курсы» — фильтрует по курсу прикрепления
  - Чекбокс «Показывать скрытые» — по умолчанию скрытые (soft-deleted) не видны
- Счётчик «Всего: N»
- Таблица: Название · Тип · Уроков · Источник · Статус · Действия

### Создать материал — это два шага

**Важно:** прикрепить материал к уроку можно только ПОСЛЕ его сохранения. Сначала создаёте сам материал, потом — на странице редактирования — добавляете уроки. Не пугайтесь, что в форме создания нет поля «Прикреплено к урокам» — оно появится после кнопки «Создать».

#### Шаг 1 — создать сам материал

1. Кнопка **«+ Добавить материал»** → попадаете на форму создания
2. Заполнить:
   - **Тип** — один из 5 (от типа зависит иконка на карточке у юзера). Тип потом нельзя поменять — если ошиблись, создайте новый и удалите старый.
   - **Название** — что увидит юзер на карточке (например: «Шаблон ABC-анализа» или «Плагин MPSTATS»)
   - **Текст кнопки (CTA)** — что написано на синей кнопке (например: «Скачать», «Открыть таблицу», «Перейти», «Открыть бот»)
   - **Описание** — короткое пояснение, опционально
   - **Источник материала** — ОДНО из двух (XOR):
     - **Внешняя ссылка (URL)** — для Google Drive, Sheets, ботов, плагинов. Просто вставить ссылку. Пример: `https://drive.google.com/file/d/.../view`
     - **Загрузить файл** — drag-n-drop ИЛИ клик «Выбрать файл». Принимаются PDF, XLSX, DOCX, CSV. Лимит 25 МБ. Файл уезжает в наш приватный Storage, юзер по клику получает временную ссылку (live на 1 час).
3. Кнопка **«Создать»** — после успешного сохранения вас автоматически перенаправит на страницу редактирования этого материала.

#### Шаг 2 — прикрепить материал к урокам

На странице редактирования (URL вида `/admin/content/materials/{id}`) внизу формы появится новый блок **«Прикреплено к урокам»**.

1. Кнопка **«Прикрепить к уроку»** открывает поиск.
2. Вводите часть названия урока ИЛИ курса (например, «ABC» найдёт уроки про ABC-анализ; «Аналитика» — уроки из соответствующего курса).
3. Кликните на нужный урок — он добавится в список прикреплений.
4. Один материал можно прикрепить к нескольким урокам сразу (типичный кейс — «Плагин MPSTATS» виден на 9 уроках).
5. Удалить привязку — кнопка `×` справа от урока в списке.

Без хотя бы одной привязки к уроку материал не виден юзерам. Не забывайте этот шаг.

### Редактировать существующий

1. Клик по названию материала в таблице → открывается форма редактирования
2. Меняете что нужно
3. В блоке «Прикреплено к урокам» — стрелочки для удаления текущей привязки или поиск + добавление новых
4. Сохранить

### Скрыть материал (soft-delete)

В строке таблицы — иконка глаза (зачёркнутый глаз = «скрыть»). Скрытый материал:
- Не виден юзерам на странице урока
- Не виден в админке по умолчанию (включить чекбокс «Показывать скрытые», чтобы найти)
- Можно вернуть обратно тем же глазом

Используйте, если материал устарел, но удалять полностью пока не хотите.

### Удалить материал полностью

Иконка корзины в строке → подтверждение → материал удаляется навсегда. Если был файл в нашем Storage — он тоже удаляется. Восстановить нельзя. Все привязки к урокам тоже исчезают.

### Найти конкретный материал

- Поиск по названию в фильтрах
- Фильтр по типу (5 чипов)
- Фильтр по курсу — выбрать в выпадашке, увидите только материалы, прикреплённые к урокам этого курса
- Колонка «Уроков» в таблице показывает, к скольким урокам привязан материал (например, «9» у плагина MPSTATS)

### Найти урок, чтобы привязать материал

При создании/редактировании в поле «Прикреплено к урокам» вводите часть названия:
- Имя курса: «Аналитика для маркетплейсов»
- Имя урока: «Сезонность спроса»

В выпадашке появятся подходящие. Клик → урок добавлен в список прикреплений. Удалить из списка — крестик рядом.

## Куда смотреть после привязки

После сохранения материала откройте на самой платформе соответствующий урок (`https://platform.mpstats.academy/learn/{lesson_id}`) — секция «Материалы к уроку» должна обновиться с новой карточкой. Если материал не появился:
- Проверьте, что материал не «Скрыт» (галочка в админке)
- Проверьте, что урок реально доступен юзеру с подпиской (на залоченном уроке секция не рендерится — это by design)
- Сообщите в чат разработке, если что-то не сходится

## Безопасность

- **Внешние ссылки** (Google Drive и т.п.) — открыты «всем по ссылке». Это компромисс с Google Drive: мы НЕ показываем секцию материалов юзерам без подписки, но саму ссылку технически может прислать кто угодно. Если переживаете за конкретный материал — лучше залить файл к нам (через «Загрузить файл»), тогда доступ строго через нашу подписку.
- **Загруженные файлы** хранятся приватно: каждый клик юзера генерирует одноразовую ссылку с TTL 1 час. Скачать файл напрямую без активной подписки нельзя.
- Материалы залоченных уроков **не утекают** — наш сервер вообще не отдаёт их в ответе на запрос урока, секции на странице нет в коде HTML.
